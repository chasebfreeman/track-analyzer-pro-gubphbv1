## Managing Tracks

### Adding a New Track

1. Go to the **Tracks** tab
2. Tap the **+** button in the top right
3. Enter the track name (e.g., "Daytona International Speedway")
4. Enter the location (e.g., "Daytona Beach, FL")
5. Tap **Add Track**

### Viewing Your Tracks

- All your tracks are displayed as cards
- Each card shows:
  - Track name
  - Location
  - Track icon
- Tap any track to start recording a reading

### Filtering by Year

- Scroll the year chips at the top
- Tap "All Years" to see all tracks
- Tap a specific year to filter tracks with readings from that year

## Recording Readings

### Starting a New Reading

1. Go to the **Record** tab
2. Tap the dropdown to select a track
3. Choose your track from the list

### Entering Data

For each lane (Left and Right), enter:

**Temperature & Conditions:**
- Track Temp (¬∞F)
- UV Index (0-11)

**Measurements:**
- Keg SL
- Keg Out
- Grippo SL
- Grippo Out
- Shine

**Additional Info:**
- Notes (any observations or comments)
- Photo (optional - tap "Add Photo")

### Adding Photos

1. Tap **Add Photo** in either lane section
2. Choose a photo from your library
3. The photo will appear as a preview
4. Tap **Change Photo** to select a different image

### Saving Your Reading

1. Fill in all required data
2. Tap **Save Reading** at the bottom
3. You'll see a success message
4. The form will clear, ready for the next reading

### Canceling

- Tap **Cancel** to clear all fields without saving
- This is useful if you want to start over

## Browsing Data

### Viewing Readings

1. Go to the **Browse** tab
2. Select a year from the top filters (or "All Years")
3. Select a track from the dropdown
4. Readings are organized by day

### Expanding Days

- Tap any day header to expand/collapse
- See all readings for that day
- Each reading shows:
  - Time recorded
  - Quick preview of left/right lane temps and UV

### Viewing Details

1. Tap any reading card
2. See complete information:
   - All measurements for both lanes
   - Photos (if uploaded)
   - Notes
   - Track information

### Deleting Readings

1. Open a reading detail
2. Tap the **Delete** button
3. Confirm deletion
4. The reading will be permanently removed

### Refreshing Data

- Pull down on any list to refresh
- This ensures you see the latest data
- Useful when working with a team

## Tips & Best Practices

### Recording Tips

1. **Be Consistent**
   - Record at the same times each day when possible
   - Use the same measurement techniques
   - Take photos from similar angles

2. **Add Context in Notes**
   - Weather conditions
   - Recent track changes
   - Unusual observations
   - Time since last race

3. **Regular Recording**
   - Record multiple times per day
   - Track changes throughout the day
   - Build a comprehensive history

### Data Organization

1. **Use Clear Track Names**
   - Include full track name
   - Add configuration if applicable (e.g., "Daytona - Road Course")

2. **Year Filtering**
   - Use year filters to focus on current season
   - Compare year-over-year trends

3. **Photo Quality**
   - Take clear, well-lit photos
   - Capture the entire lane surface
   - Consistent framing helps comparison

### Team Collaboration

1. **Shared Access**
   - All team members can view all readings
   - Everyone sees the same data in real-time

2. **Communication**
   - Use the Notes field to communicate with team
   - Add your initials to notes for clarity

3. **Data Integrity**
   - Only delete readings if absolutely necessary
   - Historical data is valuable for trends

## Troubleshooting

### Can't Log In

**Problem:** Login fails or shows error

**Solutions:**
- Check your email and password are correct
- Ensure you have internet connection
- Try resetting your password
- Contact support if issue persists

### Photos Won't Upload

**Problem:** Can't add photos to readings

**Solutions:**
- Grant photo library permission in device settings
- Check internet connection
- Ensure photo file size is reasonable (< 10MB)
- Try a different photo

### Data Not Syncing

**Problem:** Readings don't appear or update

**Solutions:**
- Pull down to refresh the list
- Check internet connection
- Log out and log back in
- Restart the app

### App Crashes

**Problem:** App closes unexpectedly

**Solutions:**
- Update to the latest version
- Restart your device
- Clear app cache (in device settings)
- Reinstall the app if needed
- Contact support with details

### Missing Readings

**Problem:** Can't find a reading you recorded

**Solutions:**
- Check you're viewing the correct track
- Try "All Years" filter
- Verify you're logged in with correct account
- Check if reading was accidentally deleted
